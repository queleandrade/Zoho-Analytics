# Dashboard

Vamos criar um dashboard com base nos gráficos, tabelas e outros elementos que já criamos.

## 1º Passo

- Clique em "+ Criar" no menu principal.
- Selecione "Painel", como mostrado na imagem abaixo:

<img width="311" alt="dashboard1" src="https://github.com/queleandrade/Zoho-Analytics/assets/75625651/fc455285-eb69-496c-81b9-dff40459acc3">

- A seguinte tela será aberta:

<img width="957" alt="dashboard2" src="https://github.com/queleandrade/Zoho-Analytics/assets/75625651/8f3eafe8-c454-4cb8-bab9-2111ca800d11">

## 2º Passo

- Para este painel, a primeira coisa a ser adicionada serão os widgets.
- Clique em "Widget" e preencha os dados conforme mostrado na imagem abaixo. Nas configurações, você pode alterar estilo, cor e outros aspectos.

<img width="956" alt="dashboard3" src="https://github.com/queleandrade/Zoho-Analytics/assets/75625651/d6dcd621-e747-4b43-ad11-037181c733de">

- Resultado do segundo passo:

<img width="954" alt="dashboard4" src="https://github.com/queleandrade/Zoho-Analytics/assets/75625651/e7f944c9-14fa-4f07-9817-c3f2911338df">

## 3º Passo

- Repita o procedimento do segundo passo para criar um widget de total de impostos e um para a meta do curso usando um gráfico de marcador.
- No gráfico de marcador, preencha os dados conforme mostrado na imagem abaixo. O campo "Destino" equivale à meta máxima, e o "Intervalo Máximo" é o limite do gráfico.

![dashboard5](https://github.com/queleandrade/Zoho-Analytics/assets/75625651/801f801a-cf37-4bf1-b000-8abcfdde711d)

- Abaixo, o nosso dashboard ficará assim:

<img width="952" alt="dashboard6" src="https://github.com/queleandrade/Zoho-Analytics/assets/75625651/5b483f2a-f464-4b9b-ad4f-9c5ef4860b0b">

- Para saber quanto já vendi de estoque de produtos, crie um widget usando o gráfico de dados de discagem completa. Preencha conforme a imagem abaixo. Nas configurações, edite estilos e outros aspectos conforme sua criatividade.

<img width="947" alt="dashboard7" src="https://github.com/queleandrade/Zoho-Analytics/assets/75625651/2608bfb3-62a9-49af-a74d-13020c8394da">

- Agora temos quatro widgets:

<img width="958" alt="dashboard8" src="https://github.com/queleandrade/Zoho-Analytics/assets/75625651/1376be45-2d75-41ca-8500-0b9e785f149d">

- Para deixar organizado, clique e arraste os widgets.
