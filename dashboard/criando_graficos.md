# Criando Gráficos

Criando gráficos no Zoho Analytics para visualizar e analisar os dados.

### Passo a Passo

1. **Abrir a Tela de Criação de Gráficos**
   - Clique em "+ Criar" no painel principal.
   - Selecione "Visualização de Mapa".

   ![image](https://github.com/queleandrade/Zoho-Analytics/assets/75625651/bf668d38-e7fd-4e58-981f-2aaea739fc7e)

2. **Criar Gráfico de Barras**
   - Escolha a tabela desejada e preencha os campos.

   <img width="958" alt="grafico2 2" src="https://github.com/queleandrade/Zoho-Analytics/assets/75625651/27c773fe-60b6-4407-b835-8389c912c785">

3. **Utilizar Filtro para Análise Específica**
   - Use o filtro para refinar sua análise e gerar gráficos mais precisos.

   <img width="950" alt="grafico3" src="https://github.com/queleandrade/Zoho-Analytics/assets/75625651/43e39dde-88fb-495d-a99c-be56354208f3">

4. **Filtro do Usuário**
   - Permita que o usuário selecione os filtros enquanto visualiza o gráfico para uma análise mais personalizada.

   ![image](https://github.com/queleandrade/Zoho-Analytics/assets/75625651/a96dcec2-6218-4cc3-aee7-c1b71866dcae)

   - Como fica a visualização para o usuário
  
   <img width="956" alt="grafico 4 4" src="https://github.com/queleandrade/Zoho-Analytics/assets/75625651/ccd9abef-5065-4dc5-89ab-c414ed9b6625">

6. **Criar Gráfico de Pizza**
   - Siga o mesmo procedimento para criar um gráfico de pizza.
